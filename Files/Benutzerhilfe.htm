1 Übersicht

Dieses Dokument beschreibt die Benutzung des Arbeitszeugnisgenerators.

Die Grundfunktion ist es, aus den in der Eingabemaske eingegebenen Daten und eine Vorlage ein Zeugnis zu generieren.

Funktionen:

·         Personendaten speichern & laden

·         Bewertungskriterien auswählen und Anpassen

·         Ein Arbeitszeugnis aus einer Vorlage generieren

 

2 Die Vorlage

Die Vorlage ist das Dokument, aus dem die Zeugnisse generiert werden. Standartweise ist eine Vorlage vorhanden, die als Beispiel verwendet werden kann.

Sie können jederzeit eine andere Vorlage auswählen.

 

2.1 Die Vorlage erstellen

Sie können die Vorlage, auf die der Zeugnisgenerator zugreifen soll, frei gestalten.

Da das Programm aber das Dokument ausließt, um Herr zu Frau und er zu sie zu machen, wenn in der Maske das Geschlecht als weiblich angegeben wird, muss das Dokument für eine männliche Person geschrieben werden.

Bei allen Wörter die mit einem X Markiert sind, wird die Groß- / Kleinschreibung beachtet

 

Damit die Vorlage von dem Programm angenommen wird, muss das Dokument mithilfe einer Excel Datei in eine Serienbriefvorlage verwandelt werden.

1.       In Microsoft Word klicken Sie dazu in der Menüleiste auf Sendungen, danach auf Empfänger auswählen und dann auf Vorhandene Liste Verwenden.

2.       Wählen sie die Daten.xlsx aus um Zugriff auf die Seriendruckfelder zu erhalten.

3.       Nun können sie unter Seriendruckfeld einfügen alle Seriendruckfelder sehen, die das Programm braucht.

Die Excel Datei bekommen Sie wenn sie in der Menüleiste unter Serienbrief auf Seriendruckfeld Datei erstellen klicken.

Sie können jedes Seriendruckfeld beliebig oft verwenden, wenn sie eines nicht brauchen, müssen sie es auch nicht verwenden.

Wenn sie bei den Seriendruckfeldern Aufgaben und Praktische Erfahrung Aufzählungszeichen setzten, werden diese auch für jeden Absatz aus den entsprechenden Feldern verwendet, wenn das Häkchen Aufzählungszeichen gesetzt ist.

 

2.2 Die Vorlage auswählen

Wenn Sie eine Vorlage auswählen wollen, klicken Sie auf Vorlage auswählen und wählen sie die Vorlage aus.

Beachten sie bitte, dass die Vorlage nach der Auswahl nicht mehr verschoben werden darf, sonst findet das Programm sie nicht mehr und greift auf die Standartvorlage zurück.

 

3 Bewertungskriterien

Die Ausgewählten Bewertungskriterien werden als Fließtext unter dem Seriendruckfeld Benotung in das Dokument gesetzt.

Die Standardkriterien sind nur Beispielkriterien mit Beispieltexten. Sie können diese für Ihren Nutzen anpassen.

 

3.1 Bewertungskriterien auswählen

Um ein Bewertungskriterium für die Erstellung des Zeugnisses auszuwählen, wählen sie im Kriteriums-Manager die Kriterien aus, die Sie verwenden wollen.

Nun können sie eine Note und eine Variante aussuchen. Der vordefinierte Text erscheint Rechts neben der Auswahl und wird sofort mit den Daten aus der Maske angepasst.

 

 

3.2 Bewertungskriterien verwalten

Um ein Kriterium zu verwalten, klicken Sie in der Menüleiste auf Kriterien Bearbeiten.

Es öffnet sich der Kriteriums-Manager. Hier sehen sie eine Liste von allen Kriterien.

Wenn sie ein Kriterium bearbeiten wollen, wählen sie es aus und drücken links auf Kriterium bearbeiten. Es öffnet sich ein Fenster wo sie:

·         Den Namen des Kriteriums ändern können

·         Neue Varianten und Texte hinzufügen können

·         Alte Varianten umbenennen oder löschen können

·         Den vordefinierten Text bearbeiten können

Der vordefinierte Text muss, sowie die Vorlage, für eine männliche Person geschrieben werden, damit das Programm alle nötigen Wörter austauscht.

Das Wort Muster wird durch den Namen der Person ersetzt, für das das Zeugnis erstellt wird

·         Herr Muster hat seine Aufträge immer gut erledigt. wird dann z.B. zu Frau Klemens hat ihre Aufträge immer gut erledigt.

 

Wenn sie ein Kriterium im nach dem Auswählen noch bearbeiten wollen, können sie rechts auf Bearbeiten klicken um sofort zu der ausgewählten Note und Variante zu springen.

 

4 Daten speichern / laden

Sie können die Daten aus der Eingabemaske, mit Ausnahme der Bewertungskriterien, speichern und jederzeit Laden. Klicken sie dazu auf Datei, Speichern( unter).

Wenn die Daten nicht gespeichert sind, werden sie vor dem Erstellen des Zeugnisses gefragt, ob sie diese Daten speichern wollen.
